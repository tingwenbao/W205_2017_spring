Exercise 1 

The content for this excercise contain:
1. loading_and_modeling folder: set up the database of this excercise
2. transforming folder: transform the rawdata before doing the analysis
3. investigations folder: query to answer the questions

To start of: git clone the repo to local directory and follow the order of loading_and_modeling-> transforming-> investigations to see the full analysis. Details to run files in each folder is also in that folder itself.
